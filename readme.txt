== Description ==
MyEventCalendar allows you to add display events to your users. It provide different customisation

== Installation ==
Install the plugin and add the [myeventcalendar] shortcode to an empty page. You can also change the theme of the calendar in the settings page

== Frequently Asked Questions ==

= Can I add events to calendar =
Yes
= Can I delete events to calendar =
Yes

= Can I import Calendar from my google account =
At the moment you can't, but this is on the roadmap
